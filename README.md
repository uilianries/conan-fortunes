#### Install

    ./install.sh
    fortune conan

**NOTE**: Super User privileges are required to copy files to /usr/share folder.

#### Requirements

- [strfile](https://linux.die.net/man/1/strfile)
- [fortune](https://linux.die.net/man/6/fortune)
